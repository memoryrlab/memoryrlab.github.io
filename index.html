Machine lEarning and Operations Research (MEmORy) Lab

The MEmORy Lab is in the College of Computer Science and Technology at the Nanjing University of Aeronautics and Astronautics. Its aim is developing automated approaches for solving all kinds of computational search problems both from academia and industry, utilizing:

Machine Learning, Data Science / Analytics, Operations Research, Evolutionary Algorithms, Computational Intelligence, Meta/Hyper-heuristics, Adaptive/Reactive/Autonomous Search, Algorithm Portfolios, Algorithm Configuration (Tuning + Control), Meta-learning, Empirical Algorithmics, Design of Experiments, Recommender Systems (Collaborative Filtering), Boolean Satisability, Constraint Programming, Combinatorial Optimisation, Data Mining (Knowledge Discovery), Expert Systems, Decision Support Systems
