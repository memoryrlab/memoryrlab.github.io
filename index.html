Machine lEarning and Operations Research (MEmORy) Lab

The MEmORy Lab is in the Division of Natural and Applied Sciences at Duke Kunshan University, China. Its aim is developing automated approaches for solving all kinds of computational search problems both from academia and industry, utilizing:

Automated Algorithm Design, Machine Learning, Automated Machine Learning (AutoML), Data Science / Analytics, Operations Research, Evolutionary Algorithms, Computational Intelligence, Meta/Hyper-heuristics, Adaptive/Reactive/Autonomous Search, Algorithm Portfolios (Selection), Algorithm Configuration (Hyper-parameter Tuning / Optimization + Control), Meta-learning, Empirical Algorithmics, Design of Experiments, Recommender Systems (Collaborative Filtering), Boolean Satisability, Constraint Programming, Combinatorial Optimisation, Data Mining (Knowledge Discovery), Expert Systems, Decision Support Systems
